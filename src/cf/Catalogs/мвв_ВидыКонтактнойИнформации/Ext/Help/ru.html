Справочник "Виды контактной информации"

Справочник служит для классификации контактной информации.

Для каждого вида контактной информации определен тип контактной информации и тип объекта, для которого данный вид контактной информации будет использоваться.

Например, для контактной информации типа Адрес для справочника Физические лица можно определить такие виды контактной информации как "Домашний адрес" или "Рабочий адрес", а для справочника Контрагенты - "Юридический адрес" или "Почтовый адрес".
