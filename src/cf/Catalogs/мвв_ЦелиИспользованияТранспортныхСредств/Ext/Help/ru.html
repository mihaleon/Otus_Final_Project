Справочник "Модели транспортных средств (ТС)"

Данный справочник предназначен для ведения списка возможных моделей транспортных средств. Реквизиты справочника используются для автоматического заполнения данных по объету имеющему тип Транспортное средство.

  • Иномарка - признак, что ТС является иномаркой.
  • Тип ТС - тип ТС (седан, пикап и т.д.). Значение является элементом справочника Типы ТС.
  • Категория  - категория ТС (A,B,C и т.д.)
  • Двигатель - информация о двигатели ТС.
  • Мощность двигателя - мощность двигателя в л.с.
  • Максимальная масса - максимальная масса ТС
  • Грузоподъемность - грузподъемность ТС
  • Количество пассажирских мест - количество пассажирских мест в ТС
  • Тип ТС для Осаго - классификация данного ТС для целей Осаго
  • Год начала выпуска - год начала выпуска ТС
  • Год окончания выпуска  - год окончания выпуска ТС
