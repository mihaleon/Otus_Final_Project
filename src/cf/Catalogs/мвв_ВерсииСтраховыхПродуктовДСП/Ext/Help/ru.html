Справочник Версии страховых продуктов

Данный справочник содержит настройки страховых продуктов, которые могут изменяться во времени, в частности тарифы. Данный справочник вызывается из справочника «Страховые продукты».

 

На закладке «Общие» указывается период действия версии страхового продукта.

На закладке «Типы объектов и риски» указываются сгруппированный перечень типов объекта страхования и рисков, которые могут быть оформлены по договору страхования:

 

На закладке «Расчет» настраиваются тарифные факторы, а также формулы расчетов страхового тарифа

 

Установив флаг «Ввести учет факторов тарифов», появляется таблицы для заполнения факторов.

Под «Факторами» понимается перечень свойств договора, объекта и т.д. на основании, которых будет формироваться тариф. Тарифные факторы могут быть сгруппированы по разделам (верхняя таблица), с настройкой отбора. Например, раздел «Постоянное использование в пределах РФ», куда входят факторы: территория преимущественного использования, периоды использования ТС, будет доступен только для  типов объектов страхования зарегистрированных в РФ.

Факторы (нижняя таблица) могут быть определенных свойств, как примитивных (число, строка, дата), так и ссылаться на определенные справочники, например, «Цвета»

 

По каждому фактору может быть настроена подсказка, которая будет видна при его вводе. Для факторов можно установить обязательность их заполнения в договоре страхования

 

На закладке «Формулы расчетов» указывается способ расчета тарифов по указанным на закладке «Факторы» факторам. Доступна как произвольная настройка расчетов (используя программный код), так и настройка с помощью таблиц базовых тарифов и поправочных коэффициентов.

 

На закладке «Формулы расчетов» формируются таблицы, верхняя – группировка по разделам, нижняя – перечень таблиц базовых тарифов и коэффициентов. Возможно указать применение одного и того же коэффициента как к разным типам объектов, так и рисков. Если отбор указан не будет, то таблица расчетов будет применяться ко всем введенным значениям, которые в ней указаны.

Формирование таблицы расчетов необходимо выполнить в два этапа. На первом этапе формируется настройка строк и столбцов таблицы. Так определяются факторы, по которым будет указываться значение расчета.

 

После этого формируется таблица по кнопке «Построить таблицу» и заполняются значения (доступно копирование соответствующего диапазона ячеек из другого табличного редактора).
