Справочник "Страховые продукты"

В конфигурации «Управление страховым брокером» данный справочник предназначен для ведения списка возможных страховых продуктов, а также их версий. В справочнике «Страховые продукты» поле «Сторонний продукт» является обязательным для заполнения, данное поле служит для отражения принадлежности страхового продукта страховой организации.  Так, например, если у брокерской организации страховой продукт КАСКО  от страховой компании РОСНО и КАСКО от страховой компании ИНГОССТРАХ, то в справочнике будет два элемента введено КАСКО (росно) и КАСКО (ингосстрах).

Справочник вызывается из меню «Страхование» - «Прямое» -  «Конструирование продуктов» - «Страховые продукты». Элементы этого справочника используются при заполнении договора страхования.

На закладке «Общие» указывается:

Поле

Заполнение

Наименование

Краткое наименование страхового продукта, такое, какое удобно пользователю

Вид

Вид продукта выбирается из предопределенного списка. Вид «Произвольный» предназначен для тех страховых продуктов, которые не являются продуктами ни «ОСАГО», ни «ОМС». От значения этого реквизита зависит форма заполнения договора страхования . 

Полное наименование

Заполняется полное наименование страхового продукта

Группа

Заполняется в том случае, если  элемент справочника объединяется в группу

Сторонний продукт

Устанавливается флаг, принадлежность этого продукта страховой компании, наименование страховой компании заполняется из справочника «Контрагенты»

Начала действия

Дата начала действия продукта

Окончание действия

Дата окончания действия продукта

 

На закладке «Значения по умолчанию» указываются значения, которые будут подставляться при формировании договора страхования по данному страховому продукту. Например, указывается, какие виды БСО будут отражаться при формировании договора по данному продукту.

На закладке «Вкладки» указываются формы ввода и отображения договоров страхования и доп. соглашений. По умолчанию  отображается универсальная форма ввода договора, но возможна настройка для конкретного продукта специализированная форма ввода, например, для продукта ОСАГО, отображается форма ввода настоящего бланка договора.

На закладке «Настройки» возможно, вести  произвольные свойства для документов по продукту, указать отнесения их к категориям, а также перечислить приложения (бумажные документы), которые прикладываются  при выписке договора страхования.

На закладке «Шаблоны» выполняется переход на настройку «Нумераторов», «печатных форм», а также шаблонов калькуляторов страхового продукта.

Для страхового продукта можно также определить тарифные факторы (факторы риска) и непосредственно тарифное руководство. Данная настройка выполняется в справочнике «Версии страхового продукта». Добавление новой версии возможно только из формы продукта на закладке «Общие».
